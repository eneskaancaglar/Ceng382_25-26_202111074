JavaScript “+” Anomalisi (String + Number)

JavaScript’te + operatörü iki işi birden yapar: sayı toplama ve metin birleştirme. Eğer operandlardan biri string ise, JavaScript diğerini de string’e çevirir ve toplamak yerine birleştirir. Bu yüzden bazen “anomali” gibi görünen sonuçlar oluşur. 

'10' + 5          // "105"   (5 sayısı string'e çevrilip birleştirilir)
10 + '5'          // "105"

'10' - 5          // 5       (- operatörü string'i sayıya çevirmeye çalışır)
Number('10') + 5  // 15      (doğru toplama)

'1' + 2 + 3       // "123"   (soldan sağa: "1"+2 => "12", sonra +3 => "123")
1 + 2 + '3'       // "33"    (1+2 => 3, sonra 3+"3" => "33")

İpucu: Eğer gerçekten toplama istiyorsan Number(x), parseInt(x, 10) veya unary +x kullan.
